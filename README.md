### Installation

1. **Clone the Repository:**

   ```sh
   git clone https://github.com/joshuasp95/omdb-api-integration.git
   ```

2. **Navigate to the Project Directory:**

   ```sh
   cd omdb-api-integration
   ```

3. **Build and Execute the Project:**

   ```sh
   ./mvnw spring-boot:run
   ```
